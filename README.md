# Sistema Web de Gestión de Compras

Proyecto académico de Ingeniería de Software orientado a la gestión de compras en una organización. Automatiza procesos como registro de solicitudes, aprobación, órdenes de compra, control de inventario, evaluación de proveedores y consulta de historial.

El sistema permite mejorar la organización de la información, reducir errores en los procesos manuales y facilitar la toma de decisiones mediante el acceso a datos estructurados y reportes.

## Funcionalidades principales
- Registro y aprobación/rechazo de solicitudes de compra
- Generación de órdenes de compra a partir de solicitudes aprobadas
- Recepción de órdenes y actualización de inventario
- Control de inventario, alertas por stock bajo y notificaciones
- Evaluación de proveedores e historial de evaluaciones
- Dashboard de análisis de costos por proveedor
- Autenticación de usuarios (registro, login, perfil)

### Módulos complementarios (vistas / ERPCompras)
- Control de contratos con alertas de vencimiento
- Historial de compras con filtros
- Exportación a Excel y PDF (historial)

### Tecnologías utilizadas
**Sprint 1**
- Backend: C#, ASP.NET Core 8 MVC
- Frontend: HTML, CSS, JavaScript, Vue.js
- Base de datos: MySQL, Microsoft SQL Server

**ERPCompras**
- Backend: Python, Flask
- Frontend: Vue.js 3, Vite
  
**Herramientas**
